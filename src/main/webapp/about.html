About Us

we have developed and implemented a Web resource supporting platform (WSMS, Web Server Management System) which can provide a unified interface for multiple websites to upload, download and manage resources professionally, and decrease the developing costs and bandwidth pressure. After investigating many solutions for server management, this project is implemented using WSMS, it is an application system based on Web, and it has to be quick when users use it to upload, download resources and manage them effectively. To reach the goal, we have to organize all the files uploaded to the server in the first place so that it can be accessed fast later.
